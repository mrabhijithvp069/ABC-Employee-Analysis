# # ABC Employee Analysis Project

## Overview
This project analyzes employee data from ABC company. It includes preprocessing, visualizations, and insights.

## Preprocessing
- Replaced height column with random values between 150–180.

## Analysis Tasks
- Team distribution and percentage split.
- Position segregation.
- Age group analysis.
- Salary expenditure by team and position.
- Age vs Salary correlation.

## Visualizations
- 5 bar charts and 1 scatterplot with trendline.

## Insights
- Most employees are in a few teams.
- 25–34 is the dominant age group.
- Salary is highest in senior roles and technical teams.
- Age and salary show a positive correlation.

## Files Included
- `ABC_Employee_Analysis.ipynb` – Jupyter Notebook with code and charts.
- `abc_employees.csv` – Dataset used for analysis.
